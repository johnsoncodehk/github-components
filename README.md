# GitHub Components

### github-pinned-repos

#### Attributes

| Name | Default | Reference |
| :-- | :-- | :-- |
| data-repos | none | ```data-repos="user/repo_1, user/repo_2"``` |
| data-column-count | ```"1"``` | ```data-column-count="2"``` |
| data-column-count-media | none | ```data-column-count-media="2: 768px, 3: 992px"``` |
